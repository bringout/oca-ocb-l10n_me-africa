# Saudi Arabia - Accounting


Saudi Arabia Accounting Module
===========================================================
Saudi Arabia Accounting Basic Charts and Localization

Activates:

- Chart of Accounts
- Taxes
- Vat Filling Report
- Withholding Tax Report
- Fiscal Positions


## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_sa
```

## Dependencies

This addon depends on:
- l10n_multilang
- l10n_gcc_invoice
