# Algeria - Accounting


This is the module to manage the accounting chart for Algeria in Odoo.
======================================================================
This module applies to companies based in Algeria.


## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_dz
```

## Dependencies

- base_vat
- account
