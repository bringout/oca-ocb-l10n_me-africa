# Egypt - Accounting


Egypt Accounting Module
==============================================================================
Egypt Accounting Basic Charts and Localization.

Activates:

- Chart of Accounts
- Taxes
- VAT Return
- Withholding Tax Report
- Schedule Tax Report
- Other Taxes Report
- Fiscal Positions
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_eg
```

## Dependencies

This addon depends on:
- account
- l10n_multilang
