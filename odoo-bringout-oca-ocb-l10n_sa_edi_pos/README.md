# Saudi Arabia - E-invoicing (Simplified)


E-invoice implementation for Saudi Arabia; Integration with ZATCA (POS)
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_sa_edi_pos
```

## Dependencies

- l10n_sa_pos
- l10n_sa_edi
