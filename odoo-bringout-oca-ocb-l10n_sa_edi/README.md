# Saudi Arabia - E-invoicing


E-invoice implementation for Saudi Arabia; Integration with ZATCA
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_sa_edi
```

## Dependencies

- account_edi
- account_edi_ubl_cii
- l10n_sa
- base_vat
- certificate
