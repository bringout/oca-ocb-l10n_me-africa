# Egypt E-Invoicing


Egypt Tax Authority Invoice Integration
==============================================================================
Integrates with the ETA portal to automatically send and sign the Invoices to the Tax Authority.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_eg_edi_eta
```

## Dependencies

- account_edi
- l10n_eg
