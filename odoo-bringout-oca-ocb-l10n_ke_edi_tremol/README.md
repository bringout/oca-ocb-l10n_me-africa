# Kenya Tremol Device EDI Integration


This module integrates with the Kenyan G03 Tremol control unit device to the KRA through TIMS.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_ke_edi_tremol
```

## Dependencies

- l10n_ke
