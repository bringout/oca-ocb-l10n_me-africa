# Türkiye - Accounting


This is the base module to manage the accounting chart for Türkiye in Odoo
==========================================================================

Türkiye accounting basic charts and localizations
-------------------------------------------------
Activates:

- Chart of Accounts
- Taxes
- Tax Report
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_tr
```

## Dependencies

- account
