# United Arab Emirates - Accounting


United Arab Emirates Accounting Module
=======================================================
United Arab Emirates accounting basic charts and localization.

Activates:

- Chart of Accounts
- Taxes
- Tax Report
- Fiscal Positions
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_ae
```

## Dependencies

This addon depends on:
- base
- account
